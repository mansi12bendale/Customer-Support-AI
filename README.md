# 🤖 AI Customer Support Insight Platform

## 📌 Overview

This project is an AI-powered system that analyzes customer support messages and generates actionable insights for businesses. It helps identify common issues, detect customer sentiment, and suggest responses for support agents.

---

## 🚀 Live Demo

🔗 Streamlit App:
https://customer-support-ai-scfyahbsgas6lieacexgkk.streamlit.app/

---


## 🎯 Features

* 📂 Ticket Categorization (Delivery, Refund, Payment, etc.)
* 😊 Sentiment Analysis (Positive, Neutral, Negative)
* 📊 Dashboard with Top Issues & Trends
* 🤖 Suggested Responses for Support Agents
* ⚡ Real-time Message Analysis

---

## 🧠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* TextBlob
* Pandas

---

## 📊 Dataset

The project uses a real-world customer support dataset from Kaggle (Twitter Customer Support dataset).

Due to large file size limitations, the original dataset is not included in this repository. A processed dataset (`tickets.csv`) is used for demonstration.

---

## 🏗️ Architecture

User Input → Streamlit App → Text Processing (TF-IDF) → ML Model → Sentiment Analysis → Output Dashboard

---

## ⚙️ Installation & Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📈 Business Impact

* Reduces customer support workload
* Improves response time
* Enhances customer satisfaction
* Provides actionable insights

---

## ⚠️ Limitations

* Basic keyword-based categorization
* Limited scalability
* No real-time streaming

---

## 🚀 Future Improvements

* Advanced NLP models (BERT/LLM)
* Multilingual support
* Real-time streaming
* Database integration

---
